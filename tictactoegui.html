Tic Tac Toe w/ Gui

by Benjamin Miranda

This was a class project where I crated a tic tac toe game with python that cant be played over sockets, which allows for two-way communitcation between two differernt programs. Playing both programs on one computer, a loop back address is used, it is usually 127.0.0.1. The two programs were both different, one for the host and one for the client, and both used the same class file to keep track of the game board. It first started as a game that was played solely in the command terminal of both individual programs/players.

The next step was to add a GUI, which was done using python's TKinter library. There are screens for each aspect of the game. From the host/join menus, the game itself, the play again screens, and the ending stat sheet screen.
